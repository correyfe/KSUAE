Построить 3D-графики. Общие требования для всех вариантов:
1. Для графиков:
− Использование ax.plot() с параметрами linewidth, color, linestyle.
− Добавление маркеров в ключевых точках.
− Подпись графика с формулой.
2. Для точечных графиков:
− Использование ax.scatter() с параметрами s (size), c (color), alpha.
265
− Цветовая кодировка по третьему параметру.
− Различные стили маркеров.
3. Для каркасных поверхностей:
− Использование ax.plot_wireframe() с настройкой rstride, cstride.
− Цветовая схема в зависимости от высоты.
− Подписи осей с единицами измерения.
4. Для сплошных поверхностей:
− Использование ax.plot_surface() с параметром cmap.
− Настройка освещения: shade=True, lightsource.
− Добавление colorbar с подписью.
5. Для комбинированных графиков:
− Сочетание минимум 3 типов графиков.
− Координатная разметка.
− Легенда с объяснением всех элементов.
